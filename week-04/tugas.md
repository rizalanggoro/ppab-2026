# Tugas Praktikum PAB (Week 4)

Buatlah aplikasi Android dengan mengimplementasikan **UI Components** seperti yang ada pada materi praktikum!

## Ketentuan Tugas

    1. Menggunakan Material 3 dan Jetpack Compose.
    2. Mengimplementasikan seluruh screen berdasarkan fitur yang terdapat pada proyek.
    3. Mengikuti struktur folder proyek yang telah disarankan pada saat praktikum untuk mempermudah pengembangan. Namun, jika memiliki struktur yang lebih sesuai dengan kebutuhan proyek, diperbolehkan untuk menggunakannya.

## Ketentuan Source Code Aplikasi

    1. Menggunakan Android Studio
    2. Menggunakan bahasa pemrograman Kotlin

## Detail Pengumpulan Praktikum

Tugas dikumpulkan dalam bentuk **link GitHub yang sama** seperti sebelumnya (dengan commit terbaru), file laporan dalam format PDF dengan nama **PPAB-04_NIM_Nama.pdf**, serta link YouTube yang menampilkan **seluruh UI aplikasi kelompok**.

Video dapat berupa screen recording sederhana tanpa editing, cukup menampilkan semua screen (menggunakan preview Jetpack Compose).
